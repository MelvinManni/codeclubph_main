Technologies That Excite Us

Being a Tech-based ecosystem, we tend to dabble in a few technologies and endeavor to guide and mentor young and expiring tech-enthusiast starting their journey with us. These technologies cover but not limited to these areas: Web development, Cloud engineering, Design.

Web Development: Making Life Easier for Yourself

Everyone wants to learn web development but only a small percentage eventually puts in the work. For front-end, we start with HTML which deals with the content and CSS that adds all the colour, but once we hit Javascript, the lights go out. Javascript is difficult to learn probably because it covers the front-end and back-end, but it is not impossible. Thankfully, frameworks such as Bootstrap (for CSS and Javascript), React, Angular and jQuery (for Javascript) have evolved over time and made life easier for us. Now a ten line code can be covered in two. Back-end development takes care of database and servers behind the scenes and converts http to real responses. A herculean task right? Luckily, frameworks like Django, Node/Express and Ruby on Rails have saved many from the likes of Javascript, PHP and Python. Now we have no excuse!

DevOps Engineer: A Great Alternative

DevOps in recent times has begun to make more waves and a lot of developers are towing towards this path. Luckily there is a roadmap to achieving this. First, learning key OS concepts such as Process Management, Sockets, I/O, Memory storage and File systems cannot be overemphasized. Also, you must educate yourself on management of servers using Linux or Ubuntu. A good grasp of the command line is vital too. The world has become a tech village which makes networking and security indispensible making concepts like OSI Model, DNS, HTTP, HTTPS important. You should also know how to setup web servers like Apache, IIS and Tomcat. Infrastructure is the most important aspect of DevOps and you should get comfortable with containers such as Docker and Kubernetes. Other concepts which come in handy include continous intergration and continuous delivery (CI/CD) tools, software monitoring and cloud provides such as AWS, Google Cloud, and Azure.

Product Design: What is that about?

Most people assume that product design is about aesthetics and graphics, but it is more complex than that. Areas of design include product design, UI, UX, graphic design, interaction design, etc. Product design is a collage of many roles including UX design, data analysis, prototyping, graphic and technical design, marketing etc. A product designer identifies and defines a problem. Then he creates prototypes which are tested for feedback before they are fine-tuned based on the feedbacks received. UX design sometimes overlaps with product design, but the latter focuses on the product. Product designers are expected to have strong UI/UX and visual design skills, wireframing and prototyping experience, research experience and excellent visual and communication skills with working knowledge on Sketch and Adobe CC. Like other product designers, you can be a problem solver too. You only need to put in the work.

AI to the World!

Artificial intelligence has been a field of interest since people questioned the ability of machines to think. In recentsl years, the progress made on this field has force a lot of professionals (and even amateurs) to explore this branch of tech. AI is in simple terms the ability of a machine to take human-like decisions without human intervention. Machine Learning, ML is a subset that uses statistical algorithms to create machines that improve based on experiences without explicit programming. This field is moving so rapidly that you need to update yourself very often. It is a web of Python and Data Science built on statistics and calculus with a tinge of psychology. Fortunately, it is too new for you to be left behind. Arm yourself with Python, R and Data Science course. Then proceed to get your nanodegree from Udacity or other platforms. If you have done this, you are not far from the promise land.
